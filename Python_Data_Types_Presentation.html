Introduction to Python Data Types

Understanding the Basics

Integer (`int`)

Represents whole numbers without any decimal points. Examples: x = 5 y = -10

Float (`float`)

Represents real numbers with decimal points. Examples: pi = 3.14159 temperature = -10.5

String (`str`)

Represents sequences of characters. Examples: name = 'John' message = "Hello, world!" paragraph = '''This is a multi-line string'''

Boolean (`bool`)

Represents True or False values. Examples: is_raining = True is_sunny = False

List (`list`)

Represents an ordered collection of elements. Examples: numbers = [1, 2, 3, 4, 5] names = ['Alice', 'Bob', 'Charlie']

Tuple (`tuple`)

Represents an ordered collection of elements. Examples: coordinates = (10, 20) colors = ('red', 'green', 'blue')

Dictionary (`dict`)

Represents a collection of key-value pairs. Examples: person = {'name': 'Alice', 'age': 30, 'city': 'New York'}

Set (`set`)

Represents an unordered collection of unique elements. Example: unique_numbers = {1, 2, 3, 4, 5}

NoneType (`None`)

Represents the absence of a value or a null value. Example: x = None
